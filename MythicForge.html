⚒ MythicForge神話工坊

視覺化編輯 MythicMobs 怪物、MythicCrucible 物品、技能與掉落表，直接讀寫伺服器設定檔。

  1. 使用 Chrome 或 Edge 開啟本頁（檔案讀寫功能需要）
  2. 點右上角「📂 開啟 MythicMobs 資料夾」，選擇伺服器的 plugins/MythicMobs
  3. 左側選擇分類與條目開始編輯，Ctrl+S 儲存目前檔案
  4. 存檔後到遊戲內或後台執行 /mm reload 讓變更生效

⚠ 儲存時只會改寫你編輯過的條目區塊，檔案中其他條目與註解會原樣保留（但被編輯條目內部的註解會遺失）。建議重要修改前先備份。
